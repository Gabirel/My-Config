# dotfiles

Here's the structure of dotfiles:
- .config
    - fcitx
- .SpaceVim.d
    - init.vim
- .wine
    - DosBox-0.74.conf
- etc
    - hosts
    - tmux.conf
    - vimrc

- scripts
    - updatehosts.sh
        This is for updating /etc/hosts
    - ledcontrol.sh
        This is for controlling LED of keyboard by `Scroll Lock`
- tools
    - viewqr
