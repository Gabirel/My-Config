# dotfiles

Here's the structure of dotfiles:

- .config
    - fcitx
    - fish
- .SpaceVim.d
    - init.vim
- .wine
    - DosBox-0.74.conf
- etc
    - hosts
    - tmux.conf
    - vimrc
- scripts
    - mojoqq.pl

        This is the script for webqq

    - updatehosts.sh

        This is for updating /etc/hosts

    - ledcontrol.sh

        This is for controlling LED of keyboard by `Scroll Lock`

- tools
    - viewqr
